# Развертывание приложения
___

1. Скомпилировать проект для создания .jar файла
2. Создать папку с проектом на сервере
3. Скопировать compose файл в указанную папку
4. Скопировать Dockerfile в указанную папку
5. Создать .env файл и прописать переменные из compose файла
6. Скопировать папку nginx в указанную папку для отображения фронтенда
7. Скомпилированные .jar файл проекта скопировать в указанную папку
8. Командой docker compose up -d --build запустить сборку и запуск проекта
